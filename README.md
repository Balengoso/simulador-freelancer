# Simulador de Orçamento Freelancer

## Descrição
Este projeto é um simulador de orçamento de serviços de TI. O usuário pode selecionar diferentes serviços e extras, e o valor total é atualizado automaticamente na tela.

## Funcionalidades
- Seleção de serviços (Landing Page, Site Institucional, API, etc.)
- Seleção de extras (Hospedagem, Domínio, SEO, etc.)
- Cálculo automático do valor total
- Atualização dinâmica do orçamento

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## Como usar
1. Abra o arquivo `index.html`
2. Selecione os serviços desejados
3. Veja o total sendo atualizado automaticamente
4. Clique em "Finalizar Orçamento"

## Estrutura do projeto
- `index.html` → estrutura da página
- `style.css` → estilização
- `script.js` → lógica do sistema

## Observações
Os valores dos serviços foram definidos para simulação de um cenário real de mercado.
